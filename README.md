# EB1 GitHub Contribution Starter Plan

This workspace is the starting point for building a GitHub contribution portfolio for EB1 purposes using the account swati-ai-lead.

## Goal
- Build simple, visible, well-documented projects.
- Keep a clean history of progress.
- Start with beginner-friendly work and grow gradually.

## Suggested folder structure
- 01-beginner-projects/ - small projects for learning and documentation
- 02-intermediate-projects/ - slightly more structured applications
- 03-advanced-projects/ - bigger portfolio pieces

## First milestone
Start with one very simple project in 01-beginner-projects/ and document it clearly.

## Step-by-step plan
1. Create a GitHub repository under swati-ai-lead.
2. Push this starter workspace to GitHub.
3. Build one beginner project at a time.
4. Add a README for each project.
5. Make small, consistent commits.
6. Keep screenshots and notes for evidence.
