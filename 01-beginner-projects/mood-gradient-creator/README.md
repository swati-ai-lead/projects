# Mood Gradient Creator

A beginner-friendly UI project that lets you build a colorful gradient card with live feedback.

## What it does
- Allows users to select two colors with color pickers.
- Displays a live gradient preview card.
- Includes mood presets and a random mood generator.
- Shows the generated CSS code for easy reuse.

## How to use
1. Open `index.html` in your browser.
2. Pick the left and right colors.
3. Type a mood message or choose a preset.
4. Click **Random mood** to generate a new palette instantly.

## Why it's unique
This project combines a UI-driven color gradient generator with expressive mood text and preset themes, making it more playful than a standard color picker demo.

## Files
- `index.html` — the interactive UI structure
- `styles.css` — visual design and responsive layout
- `script.js` — live gradient updates and preset controls

## Deploy with GitHub Pages
1. Push this project to a GitHub repository.
2. In the repo settings, open **Pages**.
3. Set the source to the `main` branch and `/root` or the branch you use.
4. Save and wait for the site URL to appear.
5. Your app will be available at `https://<username>.github.io/<repo-name>/`.

> Note: If this project is part of a larger repo, you can also deploy the `01-beginner-projects/mood-gradient-creator` folder by creating a `gh-pages` branch or using a GitHub Actions workflow to publish the folder.
